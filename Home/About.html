About

Code Converter is a free and easy-to-use VB to C# and C# to VB code converter provided for free by Telerik.

For the heavy lifting, we rely on the open source Code Converter built by ic#code. If an error occurs in conversion, we would like to ask you to report it here.

The Code Converter website is built with Kendo UI, a complete JavaScript UI component library that allows you to quickly build eye-catching, high-quality, high-performance responsive web apps using your framework of choice.
